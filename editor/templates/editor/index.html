🔧 Instalar MAX-IDE Agent

🔌

El MAX-IDE Agent conecta tu Arduino con el navegador.

¡Solo necesitas instalarlo una vez!

1
Descarga el Agent
Haz clic en el botón para descargar:
📥 Descargar MAX-IDE Agent
📦 Archivo pequeño (~50 KB)
2
Extrae y ejecuta
Clic derecho en el ZIP → "Extraer todo", luego abre la carpeta y haz doble clic en:
▶️ start_agent.bat
💡 Se abrirá una ventana negra - ¡no la cierres!
3
¡Espera la instalación automática!
La primera vez instalará Python y Arduino CLI automáticamente.
⏱️ La instalación toma menos de un minuto.
⚠️ Si pide cerrar y volver a abrir, hazlo. Es normal la primera vez.
4
Opcional: ESP32 o clones
Si usas ESP32 Dev Module: abre CMD y ejecuta arduino-cli core install esp32:esp32.
Si no detecta el puerto (clones/ESP32): instala el driver CH340 o CP2102 según tu placa.
Cierra Arduino IDE o el Monitor Serial de MAX-IDE si están usando el puerto.
🎉
¡Solo descarga, extrae y doble clic!
El instalador configura todo automáticamente.
1
Instala Homebrew (si no lo tienes)
Abre Terminal (Cmd + Espacio → escribe "Terminal") y ejecuta:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
💡 Sigue las instrucciones en pantalla. Te pedirá tu contraseña.
2
Agrega Homebrew al PATH
Cuando termine la instalación, Homebrew te mostrará una sección "Next steps" con comandos personalizados para tu Mac. Copia y ejecuta esos comandos (son 2-3 líneas que empiezan con echo y eval).
⚠️ Importante: Cierra y vuelve a abrir Terminal después de ejecutar los comandos.
💡 Verifica que funcionó ejecutando: brew --version
3
Instala Arduino CLI (y Python si falta)
En una nueva ventana de Terminal, ejecuta:
brew install arduino-cli python3
arduino-cli core update-index
arduino-cli core install arduino:avr
Para ESP32 Dev Module: arduino-cli core install esp32:esp32
Si no detecta el puerto (ESP32/clones): instala driver CP2102 de Silicon Labs. CH340 suele funcionar sin driver en Mac.
4
Descarga el Agent
📥 Descargar MAX-IDE Agent
5
Descomprime y ejecuta
Descomprime el ZIP (doble clic), luego en Terminal:
cd ~/Downloads/maxide-agent
bash start_agent.sh
⚠️ Si ves «command not found» o «syntax error near unexpected token 'elif'», ejecuta primero este comando (repara el script) y después bash start_agent.sh:
cd ~/Downloads/maxide-agent && sed -i '' 's/\r$//' start_agent.sh && sed -i '' 's/&> \/dev\/null/> \/dev\/null 2>\&1/g' start_agent.sh && bash start_agent.sh
Cierra Arduino IDE o el Monitor Serial si están usando el puerto.
✅
¡Listo! Cuando veas "Listening on http://localhost:8765", haz clic en "Verificar conexión" abajo.
1
Instala Python 3 y arduino-cli (si no los tienes)
Copia y pega en Terminal (Ctrl+Alt+T):
sudo apt install python3 python3-pip python3-venv
curl -fsSL https://raw.githubusercontent.com/arduino/arduino-cli/master/install.sh | sh
sudo mv bin/arduino-cli /usr/local/bin/
arduino-cli core update-index
arduino-cli core install arduino:avr
Ubuntu/Debian: apt. Fedora: sudo dnf install python3 python3-pip (y omite la línea apt).
Para ESP32: arduino-cli core install esp32:esp32
⚠️ NO uses snap para arduino-cli. Tiene problemas de permisos con puertos seriales.
2
Permisos para puertos serial
Ejecuta esto y luego cierra sesión y vuelve a entrar:
sudo usermod -a -G dialout $USER
3
Descarga el Agent
📥 Descargar MAX-IDE Agent
4
Ejecuta el Agent
Abre la terminal en la carpeta donde descargaste el ZIP (p. ej. Descargas o Downloads). Luego:
unzip -o maxide-agent.zip && cd maxide-agent && bash start_agent.sh
Si prefieres usar la ruta completa: Descargas (español) → cd ~/Descargas && unzip -o maxide-agent.zip && cd maxide-agent && bash start_agent.sh. Downloads (inglés) → cd ~/Downloads && unzip -o maxide-agent.zip && cd maxide-agent && bash start_agent.sh.
⚠️ Si ves «orden no encontrada» ($'\r') o «syntax error near unexpected token 'elif'», ejecuta primero (en la carpeta maxide-agent) y después bash start_agent.sh:
cd maxide-agent && sed -i.bak 's/\r$//' start_agent.sh && sed -i.bak 's/&> \/dev\/null/> \/dev\/null 2>\&1/g' start_agent.sh && bash start_agent.sh
Si no detecta el puerto (ESP32/clones): instala driver CH340 o CP2102. Cierra Arduino IDE o Monitor Serial si usan el puerto.
✅
¡Listo! Cuando veas "Listening on http://localhost:8765", haz clic en "Verificar conexión" abajo.
